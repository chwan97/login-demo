# login-demo

```bash
npm run dev
# or
yarn dev
```

本地启动，见 [http://localhost:3000](http://localhost:3000)

在线版本见 [https://lzhi.netlify.app/](https://lzhi.netlify.app)

目录结构如下
```
├── pages 页面
├── public 静态资源
├── out  产出物
├── app
│   ├── components 组件
│   ├── constants.js 常量
│   ├── store   Mobx 应用状态
│   ├── styles 样式
│   └── utills 工具类
├── jest.config.js
├── jsconfig.json
├── next.config.js next配置
├── package.json
├── README.md
└── yarn.lock
```
